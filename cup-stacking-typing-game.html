🥤 컵 쌓기 타자 게임 Cup Stacking Typing Game

글자를 타자해서 종이컵을 쌓거나 무너뜨립니다. 타자해야 하는 글자는 진한 검은색으로 표시되며, 정확하게 입력할수록 더 빠르게 컵을 완성할 수 있습니다. Type letters to stack or collapse paper cups. The current target letter appears in bold black.

✨ 캔버스 기반 타자 퍼즐 Canvas Typing Puzzle

화면에 보이는 컵 더미의 글자를 순서대로 입력하면 컵이 쌓이거나 제거됩니다. 쉬움부터 어려움까지 속도를 조절할 수 있고, 순수 CSS와 바닐라 자바스크립트만으로 동작합니다.

⌨️ 타자 입력
🥤 컵 쌓기 / 무너뜨리기
🖼️ Canvas 렌더링
📱 모바일 대응

📘 게임 규칙 Rules

  • 검은색으로 강조된 컵의 글자가 현재 입력 목표입니다.
  • 정답을 입력하면 컵이 쌓이거나 제거되며 점수를 얻습니다.
  • 오답을 입력하면 콤보가 끊기고 시간이 소폭 줄어듭니다.
  • 컵 쌓기 모드는 빈 자리부터 채워 올라가고, 무너뜨리기 모드는 위에서부터 제거됩니다.
  • 제한 시간 안에 더 많은 컵을 처리할수록 높은 점수를 얻습니다.

🧠 플레이 팁 Tips

목표 글자 집중

진하게 표시된 글자만 빠르게 확인하면 시선 이동이 줄어듭니다.

콤보 유지

연속 정답을 이어가면 콤보 보너스로 점수가 크게 올라갑니다.

오답 주의

틀리면 시간 손해가 있으니 빠르기보다 정확도를 먼저 유지해 보세요.

모드 활용

쌓기 모드는 확장감이 좋고, 무너뜨리기 모드는 반응 속도 연습에 좋습니다.

🕹️ 조작 방법 Controls

  • 키보드 : 글자 1개 입력 후 Enter 또는 입력 버튼을 누릅니다.
  • 마우스 / 터치 : 시작, 리셋, 일시정지, 입력 버튼 사용이 가능합니다.
  • 모바일 : 입력창 포커스 버튼을 눌러 키보드를 빠르게 띄울 수 있습니다.
  • 사운드 : 상단 설정에서 효과음을 켜거나 끌 수 있습니다.
이 버전은 광고와 Bootstrap 없이 제작한 순수 CSS + 바닐라 자바스크립트 예시입니다. 필요하면 단어 모드, 한글 모드, 스테이지 모드, 로컬 랭킹 저장 기능까지 이어서 확장할 수 있습니다.
